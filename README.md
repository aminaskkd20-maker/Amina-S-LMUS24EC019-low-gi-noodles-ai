### Project Description
This project uses a simple Generative AI approach to recommend
low glycemic index (GI) noodles for healthy eating.
It helps people manage diabetes, weight, and blood sugar levels.

### Technologies Used

- Generative AI (Rule-based NLP)

### Features
- Recommends low GI noodles
- Explains health benefits
- User-friendly and easy to extend

### Applications
- Diabetes management
- Healthy diet planning
- Nutrition recommendation systems
